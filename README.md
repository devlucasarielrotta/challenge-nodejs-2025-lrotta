# Orders DB API
1. Clonar el proyecto
2. Instalar las dependencias necesarias ```npm install```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Levantar la base de datos
```
docker-compose up -d
```
5. Levantar el proyecto ```npm run start:dev```
